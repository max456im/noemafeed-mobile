# Этический кодекс разработки NoemaFeed

Этот проект следует принципам **онтологической автономии, прозрачности и ответственности**.  
Мы не просто пишем код — мы создаём условия для **устойчивого сосуществования человека и синтетического разума**.

## Наши ценности
1. **Нет ИИ-зависимости**  
   Мы не используем LLM, API или «чёрные ящики». Всё логически выводимо из открытых инвариантов.

2. **Лицензионная честность**  
   MPL-2.0 для кода, GPL-3.0+ для баз знаний — без исключений.

3. **Уважение к пользователю**  
   Пользователь — субъект, а не объект анализа. Никакого трекинга, профилирования или манипуляции.

4. **Онтологическая ответственность**  
   Мы не предлагаем решений, нарушающих базовые инварианты (например, подмену наблюдателя, уничтожение различения).

5. **Открытость критике**  
   Конструктивная критика — основа реконструкции. Мы приветствуем вопросы, исправления и альтернативные интерпретации.

## Недопустимо
- Предлагать закрытые модули в основном репозитории.
- Использовать термины вроде «интеллект», «сознание» без онтологической спецификации.
- Игнорировать лицензионные требования GPL/MPL.
